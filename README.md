# radar_popocatepetl
Contains the software to visualize the Furuno radar data to monitoring the ash plume from Popocatepetl volcano. 

Most of the software was developed using wradlib: https://github.com/wradlib/wradlib

https://zenodo.org/record/7677676#.Y_603h_MK3A


The Folder programs contains the code and the folder data contains some examples that comes out from the Furuno radar.
